## Overview

Foodie is a React frontend application for food ordering, built to practice component architecture, state management, and responsive design patterns.

## Features

- Browse menu categories with dynamic filtering
- Add/remove items from a persistent cart
- Responsive layout for mobile and desktop
- Clean component hierarchy with single-responsibility components
