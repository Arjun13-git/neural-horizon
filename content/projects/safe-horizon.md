## Problem

During disasters, fragmented information and slow alert systems cost lives. Safe Horizon provides a unified platform for real-time monitoring and intelligent alerting.

## Technology Stack

**Frontend:** React, HTML, CSS, Bootstrap  
**Backend:** Python, FastAPI  
**Database:** MongoDB  
**External APIs:** USGS Earthquake API, OpenCage Geocoder, Groq AI, Fluvio

## Key Features

- Real-time seismic monitoring with automatic alert generation
- AI-powered safety recommendations based on location and disaster type
- Reverse geocoding to translate coordinates to human-readable locations
- Interactive disaster map visualization
